## 📘 Descripción

Este repositorio contiene un resumen estructurado de los principales módulos del curso de **Ethical Hacking**, abordando conceptos clave de ciberseguridad, análisis de vulnerabilidades, explotación y protección de sistemas.

El objetivo es consolidar el aprendizaje en un formato claro, organizado y accesible.

---

## 🎯 Objetivos

- Comprender las fases del hacking ético  
- Identificar vulnerabilidades en sistemas  
- Analizar técnicas de explotación  
- Aplicar medidas de seguridad  
- Desarrollar habilidades en ciberseguridad
